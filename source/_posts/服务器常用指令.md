# 服务器常用指令

Status: Linux

查看系统信息

`uname -a`

查看centos版本

`cat /etc/redhat-release`

给普通用户添加sudo权限

- Ubuntu

  `sudo usermod -aG sudo username`

- CentOS

  `sudo usermod -aG wheel username`

更改用户home目录

1. 在sudo用户shell下

```bash
usermod -d `/usr/newfolder` -u `uid` `username`
```

- `uid`: 通过 `id username` 来获取

2. 将原home目录下内容复制到新目录

```bash
sudo cp -R `/origin/home/.` `/new/home`
```
